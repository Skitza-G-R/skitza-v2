WHAT CHANGED IN V3 (FROM SHIPPED CODE)

  • 1. Unified focal shell. All three focal variants (mix / payment / session) now sit inside the same elevated white card — same radius (2xl), same shadow. The focal slot looks like the focal slot regardless of which variant fills it.
  • 2. Mix focal restored to mock spec. Compact bordered row → tall card. Tiny play glyph → "Listen" pill. Smaller title → 32px display title. Mystery "01" eyebrow → removed.
  • 3. "Book a session" tiles read as actions. Each studio avatar gets a tiny "Book →" hint below the name. Without it the row looks like a passive list of "people you know" — adding the verb cue costs nothing and makes intent obvious.
  • 4. Drag-to-reorder dropped. No grip handle, no cursor change, no keyboard alternative. Most artists have 1–3 studios so manual ordering wasn't paying for its complexity. Tiles render in server order (most recent first).

Why these three

These were the three priority items from the design-critique pass on PR #151. They keep the inbox model intact — the only thing v3 changes is making the model render consistently across all focal variants and removing affordances that nobody can see.
